# Boilerplate [Express](http://expressjs.com/) app

## Requirements
* [Node](https://nodejs.org/)

## Install
* `npm install -g gulp`
* `npm install`

## Usage
This app uses [Babel](http://babeljs.io/) to transpile ES6+ JS.

* To compile ./src -> ./dist
  * `gulp babel`

## Run
`npm start`
Runs `node ./dist/app.js`
